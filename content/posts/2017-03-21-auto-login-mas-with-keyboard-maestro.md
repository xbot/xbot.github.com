按以下步骤配置：

1. 在 Keychain 里创建一个密码项，填上 Apple ID 和密码。
2. 下载 [login_mas.kmmacros](/attachments/login_mas.kmmacros) 并导入。
3. 将 `YOUR@EMAIL.COM` 改成自己的 Apple ID 。
4. 将 `YOUR KEYCHAIN PASSWORD NAME` 改成 Keychain 里密码项的名称。
