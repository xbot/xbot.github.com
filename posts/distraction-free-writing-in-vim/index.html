goyo.vim是给vim提供专注写作模式的插件，配合markdown效果不错：

最好在单独的vim实例中使用，已发现和vim-workspace配合不好的情况。

对markdown文件自动开启goyo的配置如下：

function! s:auto_goyo()
    if &ft == 'markdown'
        Goyo 80
    else
        let bufnr = bufnr('%')
        Goyo!
        execute 'b '.bufnr
    endif
endfunction

augroup goyo_markdown
    autocmd!
    autocmd BufNewFile,BufRead * call s:auto_goyo()
augroup END
