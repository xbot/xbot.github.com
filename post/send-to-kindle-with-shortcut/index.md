# 迄今最好的发送网页到 Kindle 的快捷指令


转换成 PDF 和 FiveFilters 都不完美， `send.reabble.cn` 是迄今最好的解决方案，所以写了这个快捷指令：

[Send to Kindle](https://www.icloud.com/shortcuts/7335257de252447db732250739929ad4)

注意：

- 参照 Reabble 的说明做相关配置。
- 用 Kindle 接收邮箱创建一个联系人，并在快捷指令里选择他。
