WebSocket: Connection Header Value Must Contain Upgrade

用NGINX反向代理實現WSS後，創建連接時報錯：

Error during WebSocket handshake: ‘Connection’ header value must contain ‘Upgrade’

根據官方文檔，NGINX從1.3.13開始才支持這個特性，所以解決方法是升級到最新版。
