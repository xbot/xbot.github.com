利用依賴注入Mock PHP的內建函數

簡述

上篇文章討論了如何利用命名空間實現對PHP內建函數的mocking，本文介紹另一種實現方法——依賴注入。

出於編寫可測試代碼的需要，依賴注入是經常使用的一種技術。通過把代碼中依賴的其它數據獲取服務提取出來、和原有邏輯解耦，提高代碼的可測試性。只需mock這些依賴並注入到測試對象中即可。
