Linux下做Mac OSX安裝U盤的步驟

假設U盤對應/dev/sdb1、OSX安裝包是osx.dmg。整個過程就是提取出一系列文件，然後複制到U盤裏。

注意dmg2img的參數-p表示提取dmg的第幾個分區，似乎不同的OSX版本提取哪個分區也不一樣，判斷方法就是先執行dmg2img osx.dmg，看哪個分區提取得最慢就選哪個。
